# 🔐 Caesar Cipher GUI Tool

A modern **Python GUI application** that encrypts and decrypts text using the Caesar Cipher algorithm.
This project demonstrates basic cryptography concepts combined with a user-friendly interface built using Tkinter.

---

## 📌 Features

✔ Encrypt messages using Caesar Cipher
✔ Decrypt encrypted text instantly
✔ Clean and attractive graphical interface
✔ Supports uppercase and lowercase letters
✔ Ignores spaces and special characters
✔ Error handling for invalid input
✔ Clear button to reset fields

---

## 🧠 What is Caesar Cipher?

The Caesar Cipher is one of the oldest encryption techniques.
It works by shifting each letter in a message by a fixed number of positions in the alphabet.

### Example (Shift = 3)

HELLO → KHOOR

Encryption shifts forward, while decryption shifts backward.

## 🎯 How It Works

1. User enters a message and shift value
2. Clicking Encrypt shifts letters forward
3. Clicking Decrypt shifts letters backward
4. Result appears instantly in the output field
